Validador de WhatsApp

Sistema de automação que valida números de telefone em uma base de dados, verificando se possuem WhatsApp ativo. Processa arquivos CSV com números e gera relatórios de validação.

Python Selenium Pandas WhatsApp Web

Funcionalidades:

  • Leitura de arquivos CSV com números de telefone
  • Automação do WhatsApp Web para validação
  • Geração de relatórios com resultados
  • Interface gráfica para controle do processo
  • Tratamento de erros e retentativas
